Como activar la nube con Firebase Firestore
Sigue estos pasos una sola vez. Despues tus datos se sincronizan automaticamente.
1
Crear proyecto en Firebase
Ve a console.firebase.google.com, inicia sesion con tu cuenta Google y crea un nuevo proyecto (nombre libre, desactiva Analytics si quieres).
2
Activar Firestore Database
En el menu lateral: Firestore Database → Crear base de datos → Modo de prueba → elegir ubicacion (us-central1 o nam5).
3
Agregar app web al proyecto
En el proyecto → icono de engranaje → Configuracion del proyecto → Tus apps → agregar app Web (icono </>). Dale un nombre, NO actives Hosting. Copia el objeto firebaseConfig que aparece.
4
Pega tu configuracion aqui
Pega el objeto firebaseConfig completo en el campo de abajo.
